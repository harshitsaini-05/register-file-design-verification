# register-file-design-verification
This project presents the design and verification of a 4-bit register file with multiport read and write capabilities using SystemVerilog HDL. The register file enables simultaneous access to multiple registers, ensuring efficient data handling in modern digital systems.

Key Features:
- 4-bit wide register architecture
- Multiport read and write operations
- Synchronous design using clock signals
- Efficient parallel data access
- Comprehensive testbench for functional verification
- Detailed simulation and waveform analysis

Tools & Technologies:
- SystemVerilog HDL
- Cadence (Xcelium / SimVision) for simulation and waveform analysis

Applications:
- CPU register file design
- Microprocessors and embedded systems
- Digital signal processing (DSP) architectures

This project demonstrates fundamental and advanced concepts of digital design, including memory elements, multiplexing, timing control, and hardware verification techniques using industry-standard tools.
